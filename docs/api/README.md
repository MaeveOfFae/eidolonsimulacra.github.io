# API Documentation

This directory contains auto-generated API documentation for the `bpui` package.

## Generate Docs

```bash
make docs
```

## View Docs Locally

```bash
make docs-serve
```

Then open: <http://localhost:8000>

## Notes

- Documentation is generated with pdoc3
- Output is written to `docs/api/bpui/`
- Regenerate docs after code changes
